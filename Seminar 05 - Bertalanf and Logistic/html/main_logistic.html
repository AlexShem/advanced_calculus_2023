Логистическое уравнение с постоянным отловом

Пусть функция . Тогда получаем уравнение

Введем постоянный отлов :

У данного уравнения

  1. имется две положительные стационарные точки при . Причем, наибольший корень является устойчивым, а наименьший - нет;
  2. имеется одна неустойчивая стационарная точка при ;
  3. нет стационарных точек при . Здесь, популяция истребляется за конечно время.
c = .23;
x_0 = logspace(-1.5, 0.8, 35);
T = 2;
tspan = [0, T];

[t, x] = ode45(@(t, x) x - x.^2 - c, tspan, x_0);

figure(2);
plot(t, x, '-k');

x_star = roots([-1 1 -c]);
for k = 1 : length(x_star)
    if isreal(x_star(k))
        line([0, T], [x_star(k), x_star(k)], ...
            'Linestyle', '--', 'Color', 'red', 'LineWidth', 1.6);
    end
end
axis([0, T, 0, min(1, max(x_0))])
tl = title('$\dot{x} = x - x^2 - c$', Interpreter = 'latex', FontSize = 14);
subtitle(['$c = ' num2str(c) '$'], 'Interpreter', 'latex');
xlabel('$t$', 'Interpreter', 'latex', 'FontSize', 14);
ylabel('$x$', 'Interpreter', 'latex', 'FontSize', 14)

Логистическое уравнение с мягким отловом

Введем мягкий отлов, который попорционален текущей популяции: . Тогда уравнение примет вид

У данного уравнения имеется

  1. неустойчивая стационарная точка
  2. устойчивая стационарная точка

В стационарной точке сам отлов равен , который достигает максимума в точке , и равен при этом .

theta = .5;
x_0 = logspace(-2, 0.5, 35);
T = 2;
tspan = [0, T];

[t, x] = ode45(@(t, x) (1 - theta)*x - x.^2, tspan, x_0);

figure(3);
plot(t, x, '-k');
line([0 T], [1 - theta, 1 - theta], 'Color', 'red', 'LineStyle', '--', 'LineWidth', 1.6);
line([0 T], [0, 0], 'Color', 'red', 'LineStyle', '--', 'LineWidth', 1.6);

axis([0, T, 0, min(1, max(x_0))])
tl = title('$\dot{x} = (1 - \theta)x - x^2$', Interpreter = 'latex', FontSize = 14);
subtitle(['$\theta = ' num2str(theta) '$'], 'Interpreter', 'latex');
xlabel('$t$', 'Interpreter', 'latex', 'FontSize', 14);
ylabel('$x$', 'Interpreter', 'latex', 'FontSize', 14)
